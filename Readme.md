Lottie Specs
============

Python bindings for the Lottie specs


### Links

* Source: https://github.com/lottie/lottie-specs-python
* PyPI: https://pypi.org/project/lottie-specs/


### Building Packages

```bash
pip install build twine

python3 -m build --sdist
python3 -m build --wheel
twine upload dist/*
```
